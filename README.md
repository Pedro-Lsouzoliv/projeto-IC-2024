Esse projeto foi uma  análise de dados de EEG (Eletroencefalograma) com o objetivo de estudar os efeitos de diferentes cargas cognitivas no cérebro humano, sendo elas diferentes sequencias de numeros apresentadas aos voluntários, podendo conter 3, 5 ou 7 números de um algarismo. 
O trabalho foi desenvolvido por mim em conjunto com os Orientadores Raphael do Rosário e Amanda Amantes do Intituto de Física da UFBA. 

É necessário saber que todo o processo de criação dos scripts foram baseados no protocolo de coleta e para extrair os dados corretamente, devem ser rodados em uma ordem específica para que as marcações sejam extraídas, a rede seja filtrada e depois seja realizada a transformação dos dados em épocas menores para uso posterior no Software do Mosyn
