Esse projeto foi uma  análise de dados de EEG (Eletroencefalograma), realizada por mim com o auxílio dos meus orientadores do Instituto de Física da UFBA:Raphael do Rosário e Amanda Amantes, com o objetivo de estudar os efeitos de diferentes cargas cognitivas no cérebro humano. Para isso, Foram mostradas aos participantes, sequências numéricas que poderiam conter 3, 5 ou 7 números compostos por apenas 1 algarismo enquanto os dados eram pobtidos por um equipamento de EEG de 32 Eletrodos durante o processo. Após isso foram feitas as confecções dos códigos para transformar os dados no formato desejado, separar em épocas eremover dados com ruído muito significativo. Durante o processo são feitas utilizações de softwares auxiliares como o Mosyn para extração dos índices específicos e EEGlab para a maior parte da análise e visualização dos dados no Matlab.

É importante saber que a utilização dos scripts devem ser feitas em uma ordem específica que pretendo descrever nesse documento, assim como também pretendo melhorar-los para ter uma melhor legibilidade e também integrar alguns scripts para não ficar tão confuso o processo.
